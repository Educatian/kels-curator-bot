공개 자동 게시

Public

주간 추천 페이퍼

OpenAlex에서 JLS, IJCSCL, ETR&D, Instructional Science, Cognition and Instruction을 확인해 매주 한 편만 #academic-resources에 공개 게시합니다.

Qwen이 핵심 주장, 방법론, KELS 회원에게 유용한 이유, 토론 질문, 참여 프롬프트를 덧붙입니다.

Public

KELS Tech Signal

최근 arXiv tech paper와 GitHub repository 후보를 비교해 이번 주 더 볼 만한 항목 하나를 #academic-resources에 공개 게시합니다.

Qwen이 왜 지금 볼 만한지, 교육공학 적용, Learning Sciences 적용, 이슈테이킹 토픽, 참여 프롬프트를 정리합니다.

Public

Public

이미 시작 시간이 지난 이벤트와 시간 정보가 없는 공고는 자동 제외합니다.

Public

자동 인덱싱

설정된 공개 채널과 포럼의 새 메시지를 저장해 검색, digest, deadline, Q&A에 활용합니다.

Public

Public

무료 Nitro, 초대 링크 도배, 과도한 URL/멘션, 반복 도배 같은 명백한 스팸은 자동 삭제합니다.

포럼 제목/태그 제안

새 포럼 글이 올라오면 Qwen이 자연스러운 제목과 태그 후보를 운영자 DM으로 제안합니다.

봇이 공개 글 제목을 자동으로 바꾸지는 않습니다.

Moderator

자동 role 태깅

회원의 공개 대화 내용을 바탕으로 관심분야 role을 추론합니다. 기존 role을 우선 사용하고, 없으면 KELS: prefix의 새 role을 만들 수 있습니다.

안전장치: Admin & Facilitator, CommunicationOfficer, admin/communication 계열 role은 자동 생성/부여되지 않습니다.

현재 role이 없던 회원 61명에게는 KELS:OnboardingNeeded role이 적용되었습니다.

인덱싱 대상 채널

현재 봇은 다음 KELS 채널과 포럼을 중심으로 게시글을 인덱싱합니다.

#job_academic #job_practitioner #cfp-rfp #seminar-resource #academic-resources #study_workshop_bootcamp #conferencepresentations #newsletter #announcement

추가하면 좋은 기능 후보

1. 온보딩 질문 자동화

새 회원이 자기소개를 하면 관심 분야, 소속, 역할을 요약해 추천 채널과 role 후보를 안내합니다.

2. CFP 제출 도우미

CFP 링크를 넣으면 scope, deadline, fit, 준비 체크리스트를 Qwen이 요약합니다.

3. 주제별 미니 digest

AIED, CSCL, LA, ID/LXD 등 관심 분야별로 한 달치 게시글을 따로 요약합니다.

4. 운영자 승인 큐

자동 role 태깅, 새 role 생성, 포럼 태그 제안을 한 곳에서 승인/거절할 수 있게 합니다.
